setup-dev.sh:
  • Checks prerequisites (Python 3.12+, Node.js 18+, PostgreSQL 14+)
  • Creates Python virtual environment
  • Installs backend dependencies
  • Installs frontend dependencies
  • Sets up environment files
  • Creates and migrates database
  • Verifies setup with tests
  • Options: --skip-db, --skip-verify

test-all.sh:
  • Runs backend tests with pytest
  • Runs frontend tests with npm
  • Generates coverage reports
  • Optional linting checks
  • Options: --backend-only, --frontend-only, --with-lint

manage-services.sh:
  • Start/stop/restart services
  • Health checks for both services
  • Port availability checking
  • Process monitoring
  • Log streaming
  • Service status reporting
  • Commands: start, stop, restart, status, logs, backend, frontend
